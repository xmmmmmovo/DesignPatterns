# 工厂模式

用模板方法构建生成示例的工厂

## 使用模式与开发人员之间的沟通

建议在程序注释中和开发文档中记录所使用的设计模式的名称和意图

## 实例方法三种方式

```kotlin
/**
 * 工厂抽象类
 */
abstract class Factory {
    /**
     * 创建产品抽象方法
     * @param owner String
     * @return Product
     */
    protected abstract fun createProduct(owner: String): Product
}
```

```kotlin
/**
 * 工厂类
 */
class Factory {
    /**
     * 创建产品抽象方法
     * @param owner String
     * @return Product
     */
    fun createProduct(owner: String): Product = Product(owner)
}
```

```kotlin
/**
 * 工厂类
 */
class Factory {
    /**
     * 创建产品抽象方法
     * @param owner String
     */
    fun createProduct(owner: String): Product {
        // 这里需要自己写异常类
        throw FactoryRuntimeException()
    }
}
```
